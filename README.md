# George Convert

一个简单高效的电子书格式转换工具。

## 功能特点

- 支持多种电子书格式转换
- 批量转换功能
- 本地转换，保护隐私
- 实时转换进度显示

## 快速开始

### 前置要求

- Node.js >= 16
- Calibre

### 安装

```bash
# 克隆项目
git clone https://github.com/aproof-go/george-convert.git

# 进入项目目录
cd george-convert

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 使用方法

1. 打开浏览器访问 http://localhost:3000
2. 上传需要转换的电子书文件
3. 选择目标格式
4. 点击转换按钮
